# Excel Dashboard Project: Bike Purchase Analysis

## Overview
This project is an interactive Excel dashboard built to analyze customer demographics and bike purchase behavior. The dashboard highlights patterns across age groups, gender, income, commute distance, and purchase decisions.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas

## Key Features
- Interactive dashboard using slicers
- Analysis by gender and bike purchase status
- Age group segmentation
- Visual comparison of customer behavior
- Clean chart design with labeled axes and filters

## Business Questions Answered
- Which customer groups are more likely to purchase a bike?
- How does age influence bike purchase behavior?
- Are there visible differences by gender?
- What patterns can help target customers better?

## Files Included
- `dashboard/bike_sales_dashboard.xlsx` → final dashboard
- `images/dashboard-overview.png` → screenshot of dashboard
- `data/bike_sales_data.xlsx` → source dataset

## Key Insight
The dashboard suggests that bike purchase behavior varies across demographic groups, with some age segments and customer categories showing stronger purchase patterns than others.
